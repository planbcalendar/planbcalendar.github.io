개인정보처리방침

최종 업데이트: 2026년 7월 · PlanB Calendar (제작 DASIDA7)

PlanB Calendar(이하 "앱")는 사용자의 개인정보를 존중하며, 로컬 우선(local-first) 원칙에 따라 최소한의 데이터만 처리합니다. 본 방침은 앱이 어떤 정보를 어떻게 다루는지 설명합니다.

1. 수집·저장하는 정보

  • 일정 데이터 — 제목·시간·장소·메모·참석자 등. 사용자의 기기 로컬에 저장됩니다.
  • 계정 연결 정보 — Google·Microsoft 365·Notion 연결 시 OAuth 인증 토큰, Apple iCloud 연결 시 Apple ID와 앱 전용 암호. 동기화 목적에만 사용되며 기기에 보관됩니다.
  • 앱 설정 — 테마·강조색·위젯·출발지 등 환경설정. 기기 로컬에 저장됩니다.

2. 정보의 이용 목적

  • 연결된 캘린더 서비스와의 일정 동기화(조회·생성·수정·삭제).
  • 알림·출발 알림·위젯 등 앱 기능 제공.

3. 제3자 제공

앱은 사용자의 일정·계정 정보를 제작자나 제3자에게 전송·판매하지 않습니다. 데이터 통신은 사용자가 직접 연결한 캘린더 제공자(Google·Microsoft·Apple·Notion)와의 동기화에 한정됩니다.

4. 외부 서비스

연결된 각 캘린더 제공자 및 지도·경로 안내(출발 알림)에 사용되는 서비스는 각 제공자의 개인정보처리방침을 따릅니다. 앱은 해당 서비스에 사용자가 연결·요청한 범위의 정보만 전달합니다.

5. 데이터 삭제

계정 연결 해제 시 해당 계정의 인증 정보와 캐시된 일정이 기기에서 제거됩니다. 앱 삭제 시 로컬에 저장된 모든 데이터가 함께 삭제됩니다.
